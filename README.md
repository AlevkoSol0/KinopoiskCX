# KinopoiskCX

Расширение для быстрого перехода с Кинопоиска на плеер CX.

## Возможности

- Умная плашка на страницах фильмов и сериалов Кинопоиска для перехода на kinopoisk.cx
- Автоскрытие плашки при скролле
- Сохранение фильмов в библиотеку с тремя категориями: Главная, Позже, Просмотрено
- Пользовательские категории
- Закрепление важных фильмов
- Система оценок от 1 до 10
- Сортировка по дате, алфавиту или оценке
- Проверка на дубликаты при сохранении
- Ручная и автоматическая установка сайта-источника
- Переключение языка (русский / английский)
- Светлая и тёмная тема
- Облачное хранение данных (не удаляется при очистке истории)

## Установка (Chrome, Edge, Яндекс.Браузер)

1. Скачать ZIP-архив и распаковать
2. Открыть chrome://extensions, включить режим разработчика
3. Загрузить распакованное расширение, выбрать папку

## Блокировка рекламы

Сайт kinopoisk.cx содержит навязчивую рекламу. Рекомендуется установить одно из расширений:

- **Privacy Badger** — автоматически обучается и блокирует трекеры и рекламу, не требует настройки.
- **uBlock Origin Lite** — облегчённая версия uBlock Origin, соответствующая Manifest V3, эффективно блокирует рекламу без лишних разрешений.

Оба расширения бесплатны, с открытым исходным кодом и доступны в Chrome Web Store.

## Использование

- Кнопка "Кинопоиск" открывает главную страницу
- Кнопка "CX" появляется на страницах фильмов и сериалов Кинопоиска
- Сохранение активно только на заданном сайте-источнике
- Оценка выставляется при перемещении фильма в Просмотрено
- Настройки темы и языка в верхней части попапа

## Файлы

- popup.js
- content.js
